# marketplace_chancay_cip

Plataforma web con marketplace y capa analítica de matching que conecta empresas y territorio mediante rankings y recomendaciones explicables, usando datos mínimos y abiertos. Reduce incertidumbre, tiempo y riesgo en decisiones de inversión y localización, con enfoque MVP e implementación progresiva.

Estado actual:
Se muestran varios html que representan las vistas de la plataforma, la integración entre ellas mediante código aún está pendiente (no funcionan clics ni cambiar de página)
